Every agent uses the Uniswap v3 API to fetch live pool data — including liquidity depth, current APY, fees, and tick ranges — across hundreds of pools in parallel.
- **AI-Driven Routing & Trade Execution**
Groq AI analyzes the fetched Uniswap data and decides the best pool and route. The agent then generates a precise swap or liquidity provision transaction using Uniswap’s routing capabilities.
- **Automated Strategy Execution**
Agents continuously monitor Uniswap pools and automatically execute rebalancing, rotation, or compounding trades when better opportunities appear. All executions are routed through Uniswap’s permissionless liquidity.
- **On-chain Proof of Execution**
Every trade initiated by an agent is linked back to the original AI decision, creating full transparency from Uniswap execution → 0G attestation.

We have real on-chain transaction IDs from Sepolia testnet demonstrating actual swaps and routing performed by our agents using the Uniswap API.
